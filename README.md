## Keyboard Shortcuts

`Tab` - select next curve
`Shift + Tab` - select previous curve
`Ctrl + S` - save
`Ctrl + O` - open
`Ctrl + E` - export code
`ArrowRight` - shift view right
`ArrowLeft` - shift view left
`E` - activate "Edit" tool
`Esc` - cancel current edit
`Alt + F4` - close window
`Insert` - add new curve
`T` - activate "ChangeParamCount" tool
`Ctrl + Z` - undo
`Ctrl + Shift + Z` - redo
`Ctrl + Y` - redo
`Ctrl + Shift + Y` - undo
